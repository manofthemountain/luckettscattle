## Brand Asset Usage

All Lucketts Cattle Co. brand assets — including the “LCC” cattle brand mark, logos, wordmarks, 
site design elements, product photography, and any derivative brand styling — are proprietary 
assets owned exclusively by Lucketts Cattle Company.

These assets **may not be copied, reproduced, redistributed, or modified** under any circumstances 
without explicit written permission.

Permitted usage:
- Viewing within this repository  
- Referencing publicly (e.g., linking to the site)

Prohibited usage:
- Reproducing logos  
- Using the “LCC” cattle brand on merchandise  
- Using any Lucketts Cattle Co. imagery in other branding or commercial contexts  
- Creating modified or derivative versions of brand assets  
- Using the design, layout, or visual identity for other projects
